# Mobile App Enhancement Project Repository

## Overview

This project focused on enhancing a customer-facing mobile application by introducing key features that improved security, usability, and real-time communication while keeping development aligned with marketing goals and team capacity.

## Key Features Delivered

- **Secure Password Reset:** Implemented a secure, user-friendly password reset flow to strengthen account security and reduce friction for users who need to recover access.
- **End-to-End Order Tracking:** Added detailed order tracking so customers can see order progress from placement to delivery, improving transparency.
- **Real-Time Notifications:** Delivered push and in-app notifications for order status changes and important updates to improve engagement and reduce uncertainty.

## My Role

- **Requirements Analysis:** Gathered and analyzed business and marketing requirements to make sure features solved real customer and marketing needs.
- **User Stories & Acceptance Criteria:** Converted requirements into Agile user stories with clear acceptance criteria for development and QA.
- **Backlog & Sprint Facilitation:** Facilitated backlog grooming and sprint planning sessions to keep the team focused and priorities aligned.
- **Delivery Management:** Ensured timelines were achievable and aligned with team velocity to deliver on schedule without compromising quality.

## Results

- **Increased Trust & Engagement:** Security and notification improvements increased user trust and engagement with the app.
- **Fewer Support Inquiries:** Order tracking reduced customer support load by improving self-service transparency.
- **Timely Delivery:** Enhancements were delivered on schedule and met quality expectations.

## Next Steps

- Add automated e2e tests for the reset and tracking flows.
- Monitor metrics (CTR on notifications, support ticket volume) to quantify impact.
