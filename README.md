# Mobile App Enhancement Project

## Project Overview
This project focused on enhancing an existing mobile application to improve user experience, security, and customer engagement. The goal was to deliver high-impact features while balancing business demands with realistic Agile delivery timelines.

## Features Delivered
- Password reset functionality to enhance account security  
- Order tracking to provide real-time visibility into customer purchases  
- Real-time push notifications for order updates and user communication  

## Role & Responsibilities
- Gathered and analyzed business and marketing requirements  
- Converted requirements into Agile user stories and acceptance criteria  
- Facilitated backlog grooming and sprint planning sessions  
- Collaborated with Product, Marketing, and Engineering teams  
- Managed scope and expectations to align delivery with team capacity  

## Tools & Methodologies
- Agile / Scrum  
- Jira (or similar backlog management tools)  
- Stakeholder collaboration  
- Sprint planning & backlog refinement  

## Outcomes
- Improved customer trust and engagement  
- Reduced customer support requests  
- Delivered enhancements on time without scope creep

*This project demonstrates strong business analysis, stakeholder management, and Agile delivery skills in a mobile application environment.*
